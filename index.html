Database

Você vai aprender como estruturar uma base de dados. Vai saber quando fazer ou não uso de um ORM (Object-relational mapping).

Back-End

Você vai aprender a criar API (Aplication Programming Interface) e Microservices para sua aplicação.

Front-End

Você vai aprender a criar UI (User Interface) componentizadas, tanto Web quando Mobile para sua aplicação.
